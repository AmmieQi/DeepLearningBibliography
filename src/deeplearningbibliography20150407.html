DeepLearning.University - An Annotated Deep Learning Bibliography

Authors: Amund Tveit*, Torbjørn Morland and Thomas Brox Røst

*Email: amund@memkite.com

    Abstract

    DeepLearning.University is an annotated bibliography of recent publications (2014-) related to Deep Learning. If you have suggestions to improvements of this bibliography please send us an email or make your changes as a pull request on the corresponding git project at https://github.com/memkite/DeepLearningBibliography
